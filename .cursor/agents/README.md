# Agents Shelf (profils)

Ces fichiers servent de **prompts de référence** pour orienter des sous-agents spécialisés dans ce repo.

## Profils

- `shelf-backend.md` : API, DB, Prisma, jobs, storage, sécurité.
- `shelf-frontend.md` : UI Next.js, shadcn, design system (`DESIGN.md`), accessibilité.
- `shelf-mcp.md` : serveur MCP, tools/resources/prompts, API keys.
- `shelf-epub.md` : parsing EPUB, three-way merge, writeback OPF, dédup/soft delete.
- `shelf-qa.md` : plans de tests (unit/integration/e2e) + scénarios critiques.

## Usage suggéré

Quand tu me demandes “lancer un subagent”, je m’aligne sur ces profils pour découper le travail et garder une qualité constante.
